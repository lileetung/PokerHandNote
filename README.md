## PokerAlpha App 特色

PokerAlpha 是一款專為德州撲克玩家設計的專業工具，幫助您記錄關鍵手牌並透過 AI 進行深度分析。

### 主要功能

- **快速手牌記錄** - 直覺的介面設計讓您輕鬆輸入遊戲資訊
- **AI 專業分析** - 基於賽局理論最佳解（GTO）的深度分析
- **個人資料庫** - 安全儲存您的所有手牌記錄
- **視覺化輸出** - 圖片與文字牌譜輸出功能
